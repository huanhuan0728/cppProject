# 菜单栏显示窗口bug
发现在运行项目的时候，点击菜单会显示不出窗口，虽然在任务栏中能看到缩略图，但是并不能点击出该窗口，没有任何报错
发现问题是连接了显示器，移除显示器后就正常能够弹出窗口了
在重新连接显示器后仍能正常显示

因此可以得出解决办法是：
1. 在构建项目后不连接显示器，或者拔掉显示器再重新插回去
2. 在显示器拓展的桌面不能放任何的应用窗口，否则会触发这个bug
